# ✴️ type-guards

Use when you want only guard/assertion-focused ts-extras rules.

## Config key

```ts
typefest.configs["ts-extras/type-guards"]
```

## Flat Config example

```ts
import typefest from "eslint-plugin-typefest";

export default [typefest.configs["ts-extras/type-guards"]];
```
